# SQL Project - Student Management System
SQL project demonstrating CRUD operations and data analysis

This project demonstrates basic SQL operations including:
- Table creation
- Data insertion
- Data retrieval
- Aggregation
- Update and Delete queries

## How to Run
1. Open MySQL or any SQL tool
2. Run the SQL file
3. Execute queries step by step

## Concepts Used
- CREATE, INSERT, SELECT
- WHERE, GROUP BY
- AVG, COUNT
- UPDATE, DELETE
